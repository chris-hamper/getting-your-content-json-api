JSON API

What is JSON API?

  • Open standard
  • Helps create RESTful APIs quickly and consistently
  • Describes a consistent format for exchanging interrelated data using JSON
  • Anti-bikeshedding tool

Example JSON API Requests:


GET /articles/1 HTTP/1.1
Accept: application/vnd.api+json
					

GET /articles HTTP/1.1
Accept: application/vnd.api+json
					

GET /articles/1/author HTTP/1.1
Accept: application/vnd.api+json

Example JSON API Response


{
  "data": {
    "type": "articles",
    "id": "1",
    "attributes": {
      "title": "Example JSON API Response"
    },
    "relationships": {
      "author": {
      	"data": { "type": "people", "id": "9" }
      }
    }
  }
}

Not just for reading data


POST /articles HTTP/1.1
Accept: application/vnd.api+json

{
  "data": {
    "type": "articles",
    "attributes": {
      "title": "My New Article"
    }
    "relationships": {
      "author": {
      	"data": { "type": "people", "id": "42" }
	  }
    }
  }
}

Not just for reading data


PATCH /articles/1 HTTP/1.1
Accept: application/vnd.api+json

{
  "data": {
    "type": "articles",
    "id": "1",
    "attributes": {
      "title": "The New Title!"
    }
}
